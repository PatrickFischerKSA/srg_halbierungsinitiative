# Interaktive Lerneinheit: SRG-Halbierungsinitiative

Dieses Projekt ist eine interaktive Lernumgebung für Gymnasiast*innen.

Features:
- Halbtransparentes Vollbild-GIF als Hintergrund
- Interaktive Fragen mit Sofortfeedback
- Fortschrittsanzeige
- PDF-Export am Ende zur Abgabe auf Microsoft Teams

Start:
Öffne index.html oder lade das Repo auf GitHub Pages hoch.
